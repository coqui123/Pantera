About Database Backups

A database backup contains all your financial data including:

  • Symbol information and metadata
  • Historical price data
  • Real-time quotes
  • Company profiles
  • Portfolio holdings

Important: The backup file is a complete SQLite database that can be restored by replacing the current database file. Make sure to keep your backups in a safe location.

How to Restore a Backup

  1. Stop the Mango Data Service application
  2. Locate your current database file (usually at data/data.db)
  3. Rename or move the current database file as a backup
  4. Copy your downloaded backup file to the database location
  5. Rename the backup file to match the original database filename (data.db)
  6. Restart the Mango Data Service application

Warning: Restoring a backup will replace all current data with the data from the backup. Make sure you have a current backup before restoring an older one.
